# Smart Laptop Finder

A React application to help users find the perfect gaming laptop based on their preferences through an interactive quiz.

## Features

- Interactive quiz with 6 questions to determine user needs
- Personalized laptop recommendations from 28+ options
- Detailed product comparison table
- Product detail pages with specifications
- Dark/light theme support
- Responsive design

## Technologies Used

- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with shadcn/ui components
- **Routing**: React Router
- **Animations**: Framer Motion
- **Testing**: Vitest
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd smart-laptop-finder
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:8080`

### Build for Production

```bash
npm run build
```

### Run Tests

```bash
npm test
```
